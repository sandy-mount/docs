# Noskey

**Generate Nostr keys from the command line.** Creates nsec/npub, Bitcoin addresses, and SSH keys.

## Quick Start

```bash
npx noskey generate
```

## Output

- nsec (private key)
- npub (public key)  
- Bitcoin/Taproot addresses
- SSH public key
