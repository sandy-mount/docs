# FedBox

**A generic ActivityPub server.** Lightweight, modular foundation for federated applications.

## Overview

FedBox provides the core ActivityPub infrastructure without opinionated application logic. Build your own federated app on top.

## Features

- Full ActivityPub server-to-server protocol
- Actor management
- Activity processing
- Inbox/outbox handling
