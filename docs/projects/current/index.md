# Current

**Cross-platform Nostr client.** Simple, fast, and focused on the essentials.

## Overview

Current is a mobile Nostr client available for both iOS and Android. It provides a straightforward social experience without overwhelming complexity.

1. Download from App Store / Play Store
2. Create account or import keys
3. Follow accounts
4. Start posting
