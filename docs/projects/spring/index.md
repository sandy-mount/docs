# Spring

**Native iOS Nostr browser.** Clean, focused, Apple-quality experience.

## Overview

Spring is a native iOS Nostr client that emphasizes design quality and simplicity. It provides a polished reading and posting experience for iPhone and iPad users.

### Core Features

| Feature | Description |
|---------|-------------|
| **Feed** | Chronological timeline |
| **Profiles** | View and follow |
| **Posts** | Create and reply |
| **Media** | Image support |
| **Search** | Find users/content |
| **Notifications** | Mentions, replies |

### Simplicity Focus

Spring prioritizes:
- Clean reading experience
- Fast performance
- Intuitive navigation
- Quality over quantity
