# Freerse

**Decentralized social with Bitcoin.** Censorship-resistant messaging with Lightning payments.

## Overview

Freerse is a Nostr client focused on combining social networking with Bitcoin Lightning payments. It emphasizes privacy (no personal data required) and works well on older Android devices.

## Getting Started

1. Download from App Store or Play Store
2. Create account (generates keys)
3. Backup your keys securely
4. Connect Lightning wallet (optional)
5. Start posting and chatting

## Device Compatibility

Optimized for older devices:
- Works on older Android versions
- Efficient resource usage
- Good for budget phones
