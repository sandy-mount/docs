# Openvibe

**One app for all social networks.** Unified client for Mastodon, Bluesky, Nostr, and Threads.

## Overview

Openvibe is a unified social media client that aggregates multiple decentralized networks into a single interface. Post once, reach everywhere.

## Key Features

### Supported Networks

| Network | Protocol |
|---------|----------|
| **Mastodon** | ActivityPub |
| **Bluesky** | AT Protocol |
| **Nostr** | Nostr |
| **Threads** | ActivityPub |

### Core Features

| Feature | Description |
|---------|-------------|
| **Unified timeline** | All networks in one feed |
| **Cross-posting** | Post to multiple networks |
| **Multi-account** | Manage all identities |
| **Single sign-on** | One login for all |
