# Wavlake

**Music streaming with value-for-value.** Stream music, support artists directly with Bitcoin.

## Overview

Wavlake is a music platform where artists publish music and listeners can support them with Bitcoin micropayments (streaming sats). It combines traditional streaming with the value-for-value model from podcasting 2.0.

## Getting Started

### Artists

1. Create account
2. Upload tracks
3. Set up Lightning address
4. Promote your music
5. Receive streaming sats

### Listeners

1. Browse music (no account needed)
2. Create account to stream sats
3. Connect Lightning wallet
4. Stream and boost artists
