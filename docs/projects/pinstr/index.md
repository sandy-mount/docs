# Pinstr

**Pinterest on Nostr.** Curate and share visual collections.

## Overview

Pinstr is a decentralized Pinterest alternative built on Nostr. Create boards, organize images by interest, and discover visual content — all with your Nostr identity.

### Core Features

| Feature | Description |
|---------|-------------|
| **Boards** | Organize images by topic |
| **Pins** | Save images to boards |
| **Discovery** | Find content from others |
| **Social** | Like, comment, zap |
| **Sharing** | Share boards publicly |

### Interest Categories

Create boards for:
- Travel photography
- Food inspiration
- Art and design
- Bitcoin memes
- Music
- Movies
- Architecture
- Nature

## Social Features

- **Like boards** — Show appreciation
- **Comment** — Discuss content
- **Zap** — Send sats to creators
- **Follow** — See updates from users

## Comparison

| Feature | Pinstr | Pinterest |
|---------|--------|-----------|
| Protocol | Nostr | Proprietary |
| Identity | Your keys | Platform account |
| Data ownership | You | Pinterest |
| Censorship | Resistant | Platform rules |
| Payments | Zaps | None |
| Open source | Yes | No |

## Technical Details

- Built on Nostr protocol
- Uses image hosting (Nostr.build, Blossom)
- MIT licensed
- Open source
