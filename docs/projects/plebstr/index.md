# Plebstr

**Beautiful Nostr client for everyone.** Twitter-like experience with Bitcoin integration.

## Overview

Plebstr is a user-friendly Nostr client designed to be as intuitive as mainstream social apps. Available on iOS, Android, and desktop, it focuses on simplicity while providing full Bitcoin/Lightning integration.

### Core Experience

| Feature | Description |
|---------|-------------|
| **Feed** | Chronological timeline |
| **Posts** | Text, images, replies |
| **Profiles** | View and follow |
| **DMs** | Encrypted messages |
| **Zaps** | Lightning payments |
| **Search** | Find users and content |

### Bitcoin Integration

Built-in Lightning support:
- Send zaps to posts and profiles
- Receive sats for your content
- No external wallet required for basic use

## Privacy Features

- No email or phone required
- No tracking or analytics
- Keys generated locally
- End-to-end encrypted DMs

## Getting Started

1. Download from App Store / Play Store
2. Create new account (generates keys)
3. Backup your nsec securely
4. Start following and posting
