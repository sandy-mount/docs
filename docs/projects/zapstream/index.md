### NIP-53 Live Events

Streams are Nostr events:

```json
{
  "kind": 30311,
  "tags": [
    ["d", "unique-stream-id"],
    ["title", "My Live Stream"],
    ["summary", "Talking about Nostr"],
    ["streaming", "https://zap.stream/stream/abc"],
    ["status", "live"],
    ["starts", "1234567890"],
    ["p", "guest-npub", "relay", "host"]
  ],
  "content": ""
}
```

Benefits:
- Discoverable across Nostr clients
- Comments via Nostr events
- Portable stream metadata

## Getting Started

### For Streamers

1. **Login with Nostr** â€” Extension or nsec
2. **Get stream key** â€” From dashboard
3. **Configure OBS**:
   ```
   Server: rtmp://zap.stream/live
   Key: your-stream-key
   ```
4. **Start streaming** â€” Go live in OBS
5. **Receive zaps** â€” Watch sats flow in

### OBS Settings

Recommended settings:

| Setting | Value |
|---------|-------|
| **Encoder** | x264 or NVENC |
| **Bitrate** | 3000-6000 kbps |
| **Resolution** | 1920x1080 |
| **FPS** | 30 or 60 |
| **Keyframe** | 2 seconds |

### For Viewers

1. Visit [zap.stream](https://zap.stream/)
2. Browse live streams
3. Login to chat and zap
4. Support creators directly
